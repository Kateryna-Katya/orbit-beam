Политика конфиденциальности

Сбор и хранение данных

Orbit-Beam собирает данные только с вашего согласия. Мы используем комплекс мер безопасности, чтобы предотвратить несанкционированный доступ. Техническая информация (IP-адрес, тип браузера) используется исключительно для оптимизации работы сайта под нужды пользователей.

Цели обработки данных:

  • Предоставление доступа к экспертным материалам блога.
  • Информирование об акциях и специальных предложениях на разработку AI.
  • Обработка заявок через форму обратной связи.

Передача данных третьим лицам:

Мы не передаем ваши данные третьим лицам, за исключением случаев прямого требования законодательства ЕС/Италии или в случае необходимости выполнения условий договора (например, интеграция сторонних API сервисов по вашему запросу).
