Политика обработки персональных данных

1. Общие положения

Настоящая политика обработки персональных данных (далее — «Политика») определяет порядок и условия обработки данных, предпринимаемые проектом Orbit-Beam (далее — «Оператор»), и устанавливает меры по обеспечению безопасности этих данных в соответствии с требованиями GDPR.

1.1. Важнейшей целью Оператора является защита прав и свобод человека при обработке его персональных данных, включая защиту прав на неприкосновенность частной жизни.

1.2. Настоящая Политика применяется ко всей информации, которую Оператор может получить о посетителях веб-сайта orbit-beam.blog.

2. Основные понятия

  • Веб-сайт — ресурс, доступный по адресу orbit-beam.blog.
  • Пользователь — любой посетитель веб-сайта.
  • Персональные данные — любая информация, относящаяся к определенному Пользователю.
  • Обработка данных — любое действие с данными (сбор, хранение, удаление).

3. Данные, которые мы обрабатываем

  • Имя и фамилия;
  • Адрес электронной почты;
  • Номер телефона;
  • Обезличенные данные (cookie, IP-адрес) через сервисы Google Analytics.

4. Цели обработки

  • Установление обратной связи для консультаций по разработке ботов.
  • Направление уведомлений об инновациях и новых записях в блоге.
  • Аналитика использования платформы для улучшения UX.

5. Безопасность и сроки

Оператор принимает технические меры (шифрование, SSL) для защиты данных. Срок обработки данных неограничен, однако Пользователь может в любой момент отозвать согласие, написав на: support@orbit-beam.blog.

6. Заключительные положения

Актуальная версия Политики всегда доступна на главной странице сайта Orbit-Beam. Любые изменения вступают в силу с момента их публикации.

Этот сайт использует cookies для улучшения работы. Продолжая просмотр, вы соглашаетесь с нашей Cookie политикой.
